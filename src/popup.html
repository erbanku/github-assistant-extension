Generate Token on GitHub
Add up to 5 custom GitHub links for quick access on any GitHub page.

How to set up:

  1. Click "Generate Token on GitHub"
  2. Click the green "Generate token" button at the bottom
  3. Copy the generated token
  4. Paste it above and click "Save Token"
